# 网站统计

本主题整合 CNZZ 网站统计，并对样式进行了优化。如果需要本功能，请首先去 CNZZ 配置网站的统计，然后将网站ID添加到配置中。

## 如何配置 CNZZ

CNZZ 网址：[U-Web](https://web.umeng.com/)

注册/登录你的账户，立即使用并开始配置站点。
